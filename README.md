# ai-briefing

A fully local command-line tool that generates a news summary on any topic using Apple Intelligence (FoundationModels).

## System Requirements

- macOS 26.0 or later
- Apple Intelligence enabled on the device
- Swift 6.2 or later

## Usage

```
swift run ai-briefing "<topic>"
```

Example:

```
swift run ai-briefing "AI industry"
```

The tool searches recent news for the given topic and produces a written summary of key developments.
